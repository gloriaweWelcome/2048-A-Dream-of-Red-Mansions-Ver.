# 2048-A-Dream-of-Red-Mansions-Ver.
index.html和jekyll-gh-pages.yml程序实现了一个红楼梦版2048游戏网站。
该游戏网站使用《红楼梦》中的人物代替数字，可在“人物选择”里查看人物对应数字。
The index.html and jekyll-gh-pages.yml files implement a Dream of the Red Chamber themed 2048 game website. This game website uses characters from "Dream of the Red Mansions" to replace numbers. You can view the character-to-number mappings in the "Character Selection" section.
